# Snake Game in C

This is a simple Snake Game project created using C language.  
The project is made for beginners and first year students to understand basic programming concepts in C.

## Project Description

In this game, the player controls the snake and tries to eat the fruit.  
Whenever the snake eats the fruit, the score increases.

The game ends when the snake hits the wall.

This project helps in learning:

- Loops
- Functions
- Arrays
- Conditions
- Game logic
- User input
- Basic C programming

---

## Features

- Snake movement
- Fruit generation
- Score system
- Game over condition
- Simple controls
- Beginner friendly code
